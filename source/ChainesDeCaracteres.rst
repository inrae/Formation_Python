Chaines de Caractères
=====================

Les chaînes de caractères représentent le type de données le plus simple sous python : un ensemble de caractères liés par un index.

Chaque caractère utilise 1 octet en mémoire. 
La table de caractère définie est **UTF-8**. 

Il existe un module de traitement des chaînes de caractères sous Python enrichissant les méthodes natives de manipulation des chaînes de caractères.

Le type est défini par la nomenclature str, abréviation de string (chaîne de caractère en anglais).
