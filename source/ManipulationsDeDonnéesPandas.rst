Manipulations De Données Pandas
===============================

Pandas est une librairie Python d'analyse de données.
Elle permet de manipuler des tableaux sous forme de DataFrames (notion similaire au langage R)

Les structures utilisées en Pandas sont 
	* les **series** 
	* les **dataframes**.
