Structures
==========

Les structures sont un type avancé de Python permettant de manipuler des **ensembles de données**.

Les structures répondent à plusieurs nécessitées du développement que viendront soutenir :
	* **Les listes**
	* **Les tuples**
	* **Les ensembles**
	* **Les dictionnaires**
