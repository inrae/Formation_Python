Chargement et sauvegarde de données
===================================

En Python le chargement et la sauvegarde de données sont indispensables pour l'édition de programme de meilleure qualité.

Implémentation des commandes save et load dans le menu principal par exemple, Ecriture/Lecture "a la volée" dans une application de type client/server, etc...

Les exemples d'utilisation sont nombreux.
Les méthodes utilisées etr décrites sont :

	* *open*  : Permet d'ouvrir un fichier en lecture, ecriture ou les deux
	* *read*  : Permet de lire le contenu d'un fichier deja ouvert et accessible en lecture
	* *write* : Permet d'écrire dans un fichier deja ouvert et accessible en écriture
